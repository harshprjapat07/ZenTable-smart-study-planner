# Zentable - Teacher & Student Portal

Zentable is a simple web portal designed to make managing and submitting assignments easier. It uses a dual-login system to give teachers and students their own dedicated dashboards, managing all the data using JavaScript and a local JSON structure.

---

## 🚀 Features

### 👨‍🏫 Teacher Dashboard

* **Upload Assignments:** Quickly post new tasks for students to see.
* **Review Submissions:** Check and track all the assignments submitted by students.
* **Grading System:** Review the work and directly assign marks.

### 👨‍🎓 Student Dashboard

* **Assignment View:** See all active assignments posted by teachers.
* **Submission Portal:** Upload and submit completed tasks smoothly.
* **Grade Tracker:** Check marks and feedback given by teachers.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Data Management:** JSON (for mock data and user roles)

---

## 💻 How to Run the Project

1. Clone or download this repository to your local machine.
2. Open the project folder.
3. Open `index.html` directly in your browser, or use an extension like **Live Server** in VS Code to run it locally.
